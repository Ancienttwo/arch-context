# ArchContext v1.1 Integration Runbook

Date: 2026-06-19

## Scope

This runbook covers Sprint 3 integration surfaces:

- Notification providers: GitHub Check, Slack, generic webhook, and email.
- Model interoperability: LikeC4, Structurizr, and Mermaid exports.
- ChatGPT App GA: Cloud Metadata App, GPT Directory metadata, and local Secure MCP Tunnel guidance.

## Notification Providers

`NotificationEvent` is the only payload shape allowed past the notification boundary. It contains PR URL, review result, risk level, commit SHA, runtime version, event ID, and timestamp.

Provider configuration is opt-in. GitHub Check remains enabled by default; Slack, webhook, and email are disabled until a scoped user or installation config is stored. Slack and webhook require `secretRef`; email requires `unsubscribeUrl`.

Verification entrypoints:

- `bun test packages/notifications`
- `bun test apps/control-plane`
- `node scripts/privacy-packet-capture-audit.mjs docs/security/captures/sprint3-integrations.har.json`

## Model Interop

Native ArchContext files remain the source of truth. LikeC4 and Structurizr are export-first projections. Import is initialization-only and strips protected Native fields: evidence, verification, constraint, and intervention.

CLI entrypoints:

- `archctx export likec4`
- `archctx export structurizr`
- `archctx export mermaid`
- `archctx import likec4 --content '<dsl>'`
- `archctx import structurizr --content '<json>'`

Verification entrypoints:

- `bun test packages/adapter-likec4`
- `bun test packages/adapter-structurizr`
- `bun test packages/renderer`
- `bun test packages/cli`

## ChatGPT App GA

Remote MCP exposes metadata only: account, billing, installations, device sessions, app directory metadata, privacy disclosure, and release policy. Private repository context remains in the local runtime through the Secure MCP Tunnel.

Write tools stay disabled by default. When write mode is enabled, local confirmation is still required before ChangeSet apply.

Verification entrypoints:

- `bun test packages/mcp-cloud-metadata`
- `bun test packages/mcp-local`
- `bun test apps/chatgpt-ui`
- `archctx tunnel`

## Production Boundary

This runbook closes local and fixture verification. It does not claim live GPT App Directory listing, third-party Slack delivery, production email deliverability, or production packet capture until the corresponding external artifacts are recorded in `docs/security/captures/manifest.json`.
